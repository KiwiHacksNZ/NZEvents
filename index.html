Add an event

This site is open source. Fork the repo, add your event to events.json, and open a pull request.

1

Fork the repo

Go to kiwihacksnz/nzevents on GitHub and fork it.

2

Edit events.json

Add your event as a new entry following the existing format.

3

Open a PR

Submit a pull request. We'll review and merge it.

Example entry

{
  "name": "Your Event Name",
  "org": "Organisation",
  "category": "hackathon",
  "region": "auckland",
  "date": "2026-06-01",
  "years": "Year 9-13",
  "description": "What it is and why students should care.",
  "link": "https://example.com",
  "contact": "hello@example.com",
  "urgent": false
}
